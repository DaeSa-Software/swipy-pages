# Swipy Pages

Landing page estatica para publicar Swipy Punto de Venta con GitHub Pages.

## Publicacion en GitHub Pages

1. Crear un repositorio publico en GitHub.
2. Subir esta carpeta como contenido del repositorio.
3. En GitHub, ir a `Settings > Pages`.
4. Seleccionar `Deploy from a branch`.
5. Usar branch `main` y folder `/root`.

El archivo principal es `index.html`, sin build ni dependencias.
